TaskForge

A Trello-inspired task management application built with React and 
TailwindCSS.

Features
Create and manage tasks
Organize work with multiple columns
Task detail modal
Edit task titles and descriptions
Local storage persistence
Responsive Kanban-style interface
Tech Stack
React
TailwindCSS
JavaScript
Vite
Future Improvements
Drag and drop functionality
Due dates and labels
Multiple boards
User authentication
Real-time collaboration
Backend integration with Node.js and MongoDB
